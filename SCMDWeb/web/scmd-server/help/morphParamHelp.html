Definition of distance


Distance is defined as a weighted Euclidian distance.

Distance := the sum of ( user specified weight scores of individual parameters * ( SD-score of morphological parameter values of mutant - SD-score of user specified morphological value) )^2, where

SD-score of a value := ( the parameter value - the value of the wild type ) / ( standard deviation of the wild type distribution )
